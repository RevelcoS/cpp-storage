switch(var) statement checks var for equality with other vars.
case n: code; -> if var == n, do some code.
break -> get out of switch statement.
default: code; -> code is executed at the end of switch statement.
Example:
int day;
switch(day) {
	case 1:
		cout << "day 1";
		break;
	case 2:
		cout << "day 2";
		break;
	default:
		cout << "undefined day";
}
